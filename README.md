#  SmartAnalyzer (Expense Tracker)

A backend-based expense tracking and financial analysis system built using Java and Spring Boot. This application allows users to manage income and expenses and analyze financial data through REST APIs.

---

##  Features
- Add and manage income and expenses  
- Category-based expense tracking  
- Monthly financial analysis  
- RESTful API development for all operations  
- Backend validation and error handling  

---

##  Tech Stack
- Java  
- Spring Boot  
- Spring Data JPA (Hibernate)  
- PostgreSQL / MySQL  
- REST APIs  
- Postman (for testing)

---

##  API Endpoints
- `POST /expense` → Add expense  
- `GET /expenses` → Get all expenses  
- `DELETE /expense/{id}` → Delete expense  
- `GET /analysis` → Get monthly summary  

---

## How to Run
1. Clone the repository  
2. Open project in IDE (Eclipse / IntelliJ)  
3. Configure database in `application.properties`  
4. Run Spring Boot application  
5. Test APIs using Postman
